A production-ready Angular spreadsheet component with virtual scrolling, cell editing, and keyboard navigation.

Features Demonstrated

  • Virtual Scrolling: Smoothly handle 1,000 rows with Angular CDK
  • Cell Selection: Click to select, Shift+Click for range selection
  • Cell Editing: Double-click or press F2 to edit cells
  • Keyboard Navigation: Arrow keys, Tab, Enter to navigate
  • Undo/Redo: Ctrl+Z to undo, Ctrl+Y to redo
  • Row & Column Headers: Excel-style A, B, C... and 1, 2, 3...
  • Sample Data: Pre-populated with sales data including formulas

Try These Actions:

  • Click on any cell to select it
  • Double-click or press F2 to edit a cell
  • Use arrow keys to navigate between cells
  • Press Tab to move to the next cell, Enter to move down
  • Press Delete or Backspace to clear a cell
  • Hold Shift and click another cell to select a range
  • Right-click on cells to access context menu (cut, copy, paste, insert/delete rows and columns)
  • Right-click on column headers to insert/delete columns or auto-fit width
  • Use the Excel-like ribbon to format cells (font, colors, alignment, borders, etc.)
  • Edit a cell and press Ctrl+Z to undo the change
  • Scroll down to see virtual scrolling in action (1000 rows!)

Built with Angular {{ '20' }} + Angular CDK + TypeScript
